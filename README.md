# Chat Rooms App

A real-time chat application built with Node.js, React, MongoDB, and Socket.io.  
Users can create their own chat rooms or join existing ones, communicate in real time and use built-in chat commands.

## Features

- Real-time messaging with Socket.io  
- Create and join chat rooms  
- Chat commands support _``/help``_
- MongoDB data persistence   

## Tech Stack

- Frontend: React  
- Backend: Node.js, Express  
- Database: MongoDB  
- Real-time: Socket.io
- Styling: Tailwind  

## Setup & Run

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
